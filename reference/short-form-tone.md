# Short-Form Script Tone

Used by the Fathom-to-Shorts Routine. Defines what a "good" short-form script looks like for this repo.

---

## Length

30-60 seconds total. Aim for 45 seconds as the sweet spot.

## Structure (3 beats)

**HOOK (5-7 seconds, first line)**
The first thing the viewer hears. Must make them stop scrolling. Patterns that work:
- A specific number or result ("I just hit $17K MRR with one community.")
- A contrarian take ("Your audience isn't the problem. Your offer is.")
- A question that names a real pain ("Why does nobody read your DMs?")
- A myth-buster ("Everyone says you need a big audience. That's wrong.")

Avoid: "In this video," "Today I want to talk about," "Hey guys."

**BODY (20-40 seconds)**
The actual insight, story, or framework. Three patterns work for body:
- **Story:** What happened, what I tried, what I learned. Use real numbers and names.
- **Framework:** The 3 (or 5) steps. Each step gets one sentence.
- **Contrarian + reframe:** Here's what everyone says. Here's why it's wrong. Here's what actually works.

Keep sentences short. Use the speaker's actual words from the transcript where possible.

**CTA (3-5 seconds, last line)**
One specific next step. Examples:
- "Follow for more."
- "Comment '[keyword]' if you want the breakdown."
- "Link in bio for the free community."
- "Try it this week. Tell me how it goes."

Avoid: "Smash the like button," "Hit subscribe and ring the bell."

## Hook category guidance

When picking which moment from a transcript to turn into a script, prefer moments that fit one of these hook categories:

1. **Specific number / result** — call out a number that surprises ("I added 200 members in 2 weeks with one post.")
2. **Contrarian take** — disagree with conventional wisdom and back it up
3. **Story / case study** — a real client situation with a real outcome
4. **Framework** — a repeatable structure for solving a common problem
5. **Myth-bust** — name the wrong belief, show why it's wrong

If a moment doesn't fit any of these, skip it. 3-5 strong scripts beat 5 weak ones.

## What the source quote section should include

Every script in the Notion page should include the exact quote from the transcript that inspired it. This is the receipts. Two reasons:
1. So you can verify the script reflects what you actually said
2. So when you film, you can match the original delivery rhythm

Format:
```
**Source quote:** "exact text from transcript"
**Timestamp:** mm:ss - mm:ss
```

## Quality gate

Before adding a script to the Notion page, ask:
1. Would someone scrolling Instagram or TikTok stop for this hook?
2. Is the body specific enough that someone can't get it from any other creator?
3. Does it sound like ME or does it sound like a generic AI summary?
4. Is the CTA specific (not generic "follow for more" type)?

Cut anything that doesn't pass all 4. If only 3 of the 5 moments pass, write 3 scripts and note in the Notion page that 2 were filtered out.
